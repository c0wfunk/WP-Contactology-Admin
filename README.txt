Creates account creation and management system for Contactology. Upload entire wp-contactology-admin folder to your WordPress plugin folder and activate the plugin. Enter your Contactology Admin API key to Campaigns -> Settings in wp-admin.

Creates campaign post type that can be used to send autoresponder templates to client accounts.

See https://github.com/c0wfunk/WP-Contactology-Admin/blob/master/wp-contactology-admin/README.txt for version info

See https://github.com/c0wfunk/WP-Contactology-Admin/wiki/Instructions for instructions
